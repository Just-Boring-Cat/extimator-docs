# Profile and Preferences

Profile settings control personal details and default values for new projects.

## Profile

You can update:
- Name
- Job title
- Email address

Email changes require confirmation via magic link.

## Preferences

Defaults apply to new projects:
- Currency and locale
- Default pricing model
- Default hourly rate
- Sprint length and count
- Default risk buffer

## Security

Authentication uses magic links. There are no passwords to manage.
